# Brand assets

These files are unmodified product/site icons downloaded from official distribution endpoints for use as link identifiers in this personal viewer.

- `note.png`: note official site icon (`assets.st-note.com`)
- `x.png`: X official web app icon (`abs.twimg.com`)
- `youtube.png`: YouTube official site icon (`youtube.com`)
- `google-slides.png`: Google Slides official product image (`gstatic.com`)
- `notebooklm.png`: NotebookLM official Google Play app icon (`play.google.com` / `play-lh.googleusercontent.com`)

Preserve each asset's colors and aspect ratio. Do not use these files as generic action icons.
